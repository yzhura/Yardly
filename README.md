# Yardly — Система автоматизації для швейних брендів

**Yardly** — це спеціалізована SaaS ERP система, створена для малого та середнього швейного бізнесу (Instagram-магазини, ательє, цехи).

## Основний функціонал

- **Розумний Склад:** Облік тканин у метрах з функцією автоматичного бронювання під замовлення.
- **Управління замовленнями (OMS):** Обробка замовлень з Instagram/Etsy.
- **Логістика Нової Пошти:** Генерація ТТН через активний профіль ФОП та автоматичне відстеження статусів через Webhooks.
- **Виробничий цикл:** Призначення замовлень на кравчинь (Tailors), контроль дедлайнів пошиття.
- **ПРРО Checkbox:** Фіскалізація чеків (автоматично або за запитом).

## Архітектура

Проєкт побудований за принципом Multi-tenancy (один сервер — багато ізольованих клієнтів).

### Моделі даних:

1. **Tenant** — Бізнес-юніт.
2. **User** — Співробітник (Адмін/Менеджер).
3. **IntegrationProfile** — Налаштування API НП та Checkbox для конкретного ФОП.
4. **Material** — Тканини та фурнітура.
5. **Product & ProductVariant** — Каталог виробів (Колір/Розмір).
6. **Order & OrderItem** — Замовлення та його склад.
7. **OrderStatusHistory** — Лог усіх змін статусів замовлення.
8. **Tailor** — База кравчинь.
9. **Warehouse & Stock** — Облік готової продукції на складах.
